# CCNP ENARSI (300-410) 
Repositorio de resúmenes técnicos para preparación del examen ENARSI

Este repositorio contiene **23 capítulos completamente resumidos**, diseñados para ofrecer una referencia clara, directa y altamente técnica sobre los temas esenciales del examen **Cisco CCNP ENARSI (300-410)**.  
Cada capítulo presenta explicaciones concisas, conceptos clave, comandos, procesos internos, y notas de troubleshooting basadas en escenarios reales.

Los resúmenes están orientados tanto a estudio personal como a su uso en laboratorios y preparación intensiva para certificación.

## Objetivo del Proyecto

El objetivo de este repositorio es centralizar **material de estudio confiable**, directo y accionable para cualquier profesional que esté preparando el examen ENARSI.  
Los capítulos están estructurados para:

- Facilitar repasos rápidos  
- Ofrecer explicaciones profundas  
- Servir como referencia en laboratorios (GNS3, EVE-NG, CML)  
- Apoyar a instructores y estudiantes en programas de formación profesional  

---

## Cómo usar este repositorio

1. Selecciona el capítulo según el tema que deseas repasar.  
2. Repite los comandos en tu laboratorio para reforzar conceptos.  
3. Usa los apartados de troubleshooting para identificar patrones en fallos reales.  
4. Combina estos resúmenes con prácticas de configuración y tickets de fallo.
